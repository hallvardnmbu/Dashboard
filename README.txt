Project utilizing the 'StreamSync' framework for interactive use of BarentsWatch and Frost APIs.

The project is a part of the course 'IND320' at NMBU. The web app allows the user to download and
explore data, in addition to some simple interactive forecasting.

IMPORTANT
---------
* Modify the API credential paths at the top of the 'functions.fetching'-file.
* Must have Cassandra running.

USAGE
-----
* (Cassandra must be running.)
* Run the following command:
  -> streamsync run Dashboard/
